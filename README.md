# Aneefat Kufre Balogun · Portfolio

A sleek, animated personal portfolio website for **Aneefat Kufre Balogun** — a certified cloud engineer merging fashion and technology. The design follows a refined **Lemon Chiffon & Ultraviolet** palette, with smooth letter animations, floating shapes, and a moving gradient background.

🔗 **Live Preview**: [Open in browser](#) *(run `index.html` locally)*  
🐙 **GitHub Profile**: [Anny030](https://github.com/Anny030)

---

## ✨ Features

- **Letter‑by‑letter animated headlines** – each character rises with a soft blur and staggered delay  
- **Dynamic moving background** – a slow, shifting gradient combined with floating abstract shapes  
- **Fully responsive** – adapts seamlessly to mobile, tablet, and desktop  
- **Dedicated pages**:
  - **Home / Hero** – introduction, avatar, cloud & fashion focus  
  - **Projects** – filterable grid of real GitHub repositories (ChicPeek, EcoStyle, ColorCue, etc.)  
  - **About** – personal story with animated text and a direct LinkedIn call‑to‑action  
  - **Connect** – minimalist page centered on LinkedIn connection  
- **Consistent navigation & footer** across all sections  
- **GitHub integration** – links to actual repositories of the user `Anny030`

## 🚀 Getting Started

1. **Clone or download** the repository  
2. Open any of the HTML files in a modern browser (Chrome, Firefox, Edge, Safari)  
3. No build step or server required – it's plain static HTML/CSS/JS  

To edit content:
- Replace placeholder LinkedIn/GitHub URLs in the `<a>` tags and JavaScript variables (`GITHUB_USER`, `LINKEDIN_URL`)
- Update project data array inside each page’s `<script>` block

## 🔧 Customization

- **Change name / bio**: Edit the text inside the respective HTML files.  
- **Update projects**: Modify the `projectsData` array in the projects page script.  
- **Adjust animation speed**: Tweak `animation-duration` and `animation-delay` in the CSS.  
- **Replace avatar**: The avatar is an `A·B` monogram inside a circle. To use a real photo, replace the `.avatar-initials` with an `<img>` tag.
